# 🕉 Инструменты ШЧСМ — для служения и проповеди

Открытые инструменты миссии Шри Чайтанья Сарасват Матха для работы со словом
Гурудева и ачарьев. Всё считается **у вас на компьютере** — бесплатно, без
интернета (нужен один раз при установке). Ничего никуда не уходит.

## Что здесь

### 🎙 Транскрибатор ШЧСМ
Кладёте аудио (лекцию, голосовое, запись) — получаете готовый текст со знаками
препинания и **правильными именами ачарьев**. Встроен словарь по «Жемчужине
мудрости Востока» Шрилы Шридхара Махараджа.

### ✍️ Диктовщик ШЧСМ
Диктуете голосом — текст печатается сразу, с тем же выверенным словарём.

## Скачать

👉 **[Последняя версия — на странице Releases](../../releases/latest)**

- `Транскрибатор-ШЧСМ.zip` — распакуйте, откройте папку своей системы (mac / windows),
  запустите «Установить».
- `Диктовщик-ШЧСМ.zip` — аналогично.

Инструкция «в три шага» — внутри каждого пакета (файл «ЧИТАЙ-МЕНЯ»).

## Канон-принцип

**Слово Гуру машиной не правится.** Программа только помогает распознать; где она
могла ошибиться — оставляет пометку для вычитки, но текст не меняет. Последнее
слово всегда за живой вычиткой преданного.
